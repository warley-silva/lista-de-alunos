# lista-de-alunos
Projeto para a disciplina de Web II. 

## Pré-requisitos

### express
### sequelize
### ejs
### express-ejs-layouts

## Instalação

`npm install express ejs express-ejs-layouts sequelize`
e `npm install -g nodemon`

## Rodar o app no terminal

`nodemon server.js`

## No navegador use:

`localhost:8080/home`
ou
`127.0.0.1:8080/home`
